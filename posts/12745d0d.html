时隔多日我又来更新每周大学习了，这回是关于 CSS 的，CSS 是网页设计中很重要的一块，CSS 可以让网页更加美观，更加符合用户的需求，CSS 的语法也相当简单，今天就一起学习下 CSS 吧！

CSS 是什么

  • CSS 指的是层叠样式表 (Cascading Style Sheets)
  • CSS 描述了如何在屏幕、纸张或其他媒体上显示 HTML 元素
  • CSS 节省了大量工作。它可以同时控制多张网页的布局
  • 外部样式表存储在 CSS 文件中
    如果说 HTML 是身体，那么 CSS 就是衣服
    CSS 是一种描述 HTML 文档样式的语言
